--- Клавиши эмулятора ---

F11 + Esc:      Выход
F11 +  F1:      Сброс
F11 +  F5:      Старт/стоп магнитофона
F11 +  F6:      Перемотка ленты, 10 сек вперед
F11 +  F7:      Перемотка ленты, 10 сек назад
F11 +  F8:      Показать текущую позицию ленты
F11 +   9:      Громкость -
F11 +   0:      Громкость +
F11 +   M:      Звук, вкл/выкл
F11 +   F:      Полноэкранный режим, вкл/выкл
F11 +   A:      Коррекция соотн. сторон (4:3), вкл/выкл
F11 +   G:      Черно-белый режим, вкл/выкл
F11 +   S:      Имитация строк развертки ЭЛТ, вкл/выкл

--- Клавиши Поиска ---

F4:             Щелчки динамика ПК по нажатию клавиш (вкл/выкл)
Ctrl (правый):  Кириллица/латиница

Прим.
- BIOS Поиска и некоторые программы ипользуют обозначения:
  ВК (Enter), AP2 (Esc), УПР (Ctrl).
- В режиме кириллицы отображается синий фон.

--- Запуск программ ---

Могут быть загружены программы в формате BIN (двоичный образ программы),
CART (двоичный образ картриджа ПЗУ), либо WAV (звуковой файл,
представляющий ленту магнитофона).
Файлы могут быть переданы эмулятору через стандартную функцию
проводника "Открыть с помощью...", либо как параметры командной строки.

- Файл в формате BIN будет запщен автоматически.

- Файлы в формате WAV необходимо заргужать реальном времени
  с помощью функции BIOS Поиска, пункт меню "F1 - Работа с кассетой".
  Следует ввести название файла (заглавными буквами); по нажатию "ВК" (Enter)
  "магнитофон" эмулятора будет запущен автоматически.

- Файл в формате CART может быть запущен из меню BIOS,
  пункт "F2 - Работа с ПЗУ".
